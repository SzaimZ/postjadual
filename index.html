🔔 Setup OneSignal (Notifikasi Lock Screen)

⏳ Memeriksa status OneSignal...
App ID boleh dijumpai di OneSignal Dashboard → Settings → Keys & IDs
📖 Cara Daftar & Setup OneSignal (klik untuk buka)
1
Daftar akaun OneSignal
Pergi ke onesignal.com → klik "Get Started Free" → daftar dengan email atau Google
2
Buat App baru
Selepas log masuk → klik "New App/Website"
• App Name: PostJadual (atau apa-apa nama)
• Pilih platform: Web
• Klik "Next: Configure Your Platform"
3
Konfigurasi Web Platform
Pilih "My site is not fully HTTPS" → TIDAK perlu
Pilih "Site URL": masukkan URL GitHub Pages anda
Contoh: https://username.github.io
• Label: PostJadual
• Klik "Save & Continue"
4
Download SDK files
OneSignal akan bagi 2 fail untuk download:
• OneSignalSDKWorker.js
• OneSignalSDKUpdaterWorker.js
Upload kedua-dua fail ini ke root folder GitHub Pages anda (sama lokasi dengan index.html)
5
Salin App ID
Klik "Finish" → Pergi ke Settings → Keys & IDs
Salin nilai OneSignal App ID
Paste ke dalam kotak di atas → klik Simpan
6
Aktifkan notifikasi dalam app
Selepas simpan App ID → klik butang "Aktifkan Notifikasi" yang akan muncul di bawah.
iPhone akan tanya kebenaran → tap Allow
✅ Notifikasi kini akan keluar di lock screen!
